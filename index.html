Arrays - Introduction

Beginner
❤

An array is a contiguous block of memory that stores elements of the same type. Arrays provide constant-time access by index (O(1)) and are efficient for iteration.

Flowchart

graph LR
    A[Start] --> B[Declare array]
    B --> C[Initialize elements]
    C --> D[Iterate over array]
    D --> E[Access / Process elements]
    E --> F[End]

Complexity

  • Access by index: Time O(1)
  • Search (linear): Time O(n)
  • Insert/Delete (middle): Time O(n)
  • Space: O(n)

Code Examples

#include <iostream>
using namespace std;
int main(){
  int arr[5] = {1,2,3,4,5};
  for(int i=0;i<5;i++) cout << arr[i] << " ";
  return 0;
}

arr = [1,2,3,4,5]
for x in arr:
    print(x, end=' ')
